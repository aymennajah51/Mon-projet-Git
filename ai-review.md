# AI Usage Review

## Code généré par IA
Une fonction JavaScript simple pour effectuer une addition :

function addition(a, b) {
    return a + b;
}

## Modifications apportées
- Vérification de la logique de la fonction
- Ajout d’un test avec console.log
- Adaptation pour qu’elle soit simple et compréhensible

## Pourquoi Git est essentiel avec l’IA ?

- Permet de tracer les changements générés par IA
- Facilite la relecture et la correction du code
- Permet de revenir en arrière en cas d’erreur
- Garantit la qualité et la fiabilité du projet

## Conclusion
L’IA aide à générer du code rapidement, mais Git est indispensable pour assurer la qualité, la traçabilité et la gestion des modifications.
