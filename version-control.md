# Version Control

## Qu'est-ce que le version control ?

Le version control (contrôle de version) est un système qui permet de suivre les modifications d’un fichier ou d’un projet au fil du temps.  
Il permet de sauvegarder différentes versions et de revenir à une version précédente si nécessaire.

## Pourquoi on en a besoin ?

Le version control est important pour plusieurs raisons :

- Il permet de garder un historique des modifications
- Il facilite le travail en équipe
- Il permet de revenir en arrière en cas d’erreur
- Il aide à organiser et gérer le code efficacement

## Exemple

Git est un outil de version control très utilisé.
